🏃‍♂️ Nike Cinematic Website

A bold, high-performance multi-page website inspired by Nike’s design philosophy, focused on immersive storytelling, product showcase, and modern UI aesthetics.
🚀 Overview
This project is a visually engaging website built using **pure HTML and CSS**, designed to reflect the energy, motion, and innovation associated with Nike. It delivers a cinematic user experience with strong visuals, smooth layouts, and a consistent dark-themed design system.
✨ Features
* 🎬 Cinematic Homepage with hero sections and product highlights
* 🧭 Multi-page Navigation (12 fully linked pages)
* 🛍️ Product Categories – Running, Training, Racing, Lifestyle
* 🏢 Company Pages – About, Careers, Press, Sustainability
* 🛠️ Support Pages – Help Center, Shipping, Returns, Size Guide
* 🎨 Consistent UI/UX with modern typography and layout
* ⚡ Fast Performance (No JavaScript frameworks used)
* 📱 Responsive Design Structure


🛠️ Technologies Used

 HTML5
 CSS3 (Flexbox, Grid, Animations)
